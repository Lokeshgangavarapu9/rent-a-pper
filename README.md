# RentForPeer

## 🚀 Project Overview
RentForPeer is a platform designed to facilitate the sharing and renting of skills, services, and items among students within a university or college environment. It aims to create a vibrant community where students can easily find help, offer their expertise, or rent out personal items to their peers. The platform features a dashboard with sections for home, posts, explore, notifications, messages, follow, charts, and profile, providing a comprehensive experience for users.

## ✨ Features
- **User Dashboard:** A central hub for all activities, including an overview of rents, work, and active members.
- **Posts Section:** Allows users to share their services, updates, or experiences, fostering community interaction.
- **Explore Section:** Discover professional (social) and casual services offered by peers, with search and filtering capabilities.
- **Sidebar Navigation:** Easy access to different sections of the platform.
- **Responsive Design:** (Assumed based on modern web practices and general project scope, though not explicitly verified in `index.html`)

## 🛠️ Technologies Used
- **Frontend:** HTML5, CSS3, JavaScript
- **Styling:** Font Awesome for icons, Google Fonts (Roboto)

## ⚙️ Installation
To get a local copy up and running, follow these simple steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Lokeshgangavarapu9/rent-a-pper.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd rent-a-pper
   ```
3. **Open `index.html` in your browser:**
   Simply open the `index.html` file using your preferred web browser to view the application.

## 💡 Usage
- **Home:** View daily rents, work, and active members. Switch between 'Rent' and 'Work' tabs to see rented members and your work.
- **Posts:** Create new posts to offer services or share updates. View posts from other users.
- **Explore:** Search for services, people, or posts. Discover professional and casual offerings.
- **Sidebar:** Use the sidebar to navigate between different sections of the dashboard.
